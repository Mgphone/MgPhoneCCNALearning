Packet Tracer Intro

Introduction to Packet Tracer

Before configuring devices via the Command Line Interface (CLI), you must understand how to physically build networks. This lab focuses on identifying, placing, renaming, and cabling network infrastructure devices within the Cisco Packet Tracer environment.

The Components of a Network

Enterprise networks are composed of various specialized devices. Packet Tracer provides software simulations of real-world Cisco hardware to help you practice building these architectures.

🌐

Routers (e.g., Cisco 2911)

Operate at Layer 3. They connect different networks together and make path-determination decisions to route traffic across the internet.

▤

Switches (e.g., Cisco 2960)

Operate at Layer 2. They provide the physical ports to connect end devices (PCs, Servers) together to form a single Local Area Network (LAN).

🔒

Firewalls (e.g., Cisco ASA 5505)

Security appliances that monitor and control incoming and outgoing network traffic based on predetermined security rules.

Lab Task Guide

Check off the steps below as you successfully place, rename, and connect the devices in Packet Tracer. Click a task to expand instructions on how to perform it within the Packet Tracer interface.

To place a router in the logical workspace:

  1. Look at the bottom-left corner of the Packet Tracer window.
  2. Click on the Network Devices icon (it looks like a router/switch stack).
  3. In the sub-menu below it, ensure Routers is selected.
  4. Find the model 2911 in the list of devices on the right.
  5. Click and drag two 2911 routers into your workspace.

To place a switch:

  1. In the bottom-left Network Devices menu, click the Switches icon.
  2. Find the model 2960 in the device list.
  3. Click and drag two 2960 switches into your workspace.

To place a firewall security appliance:

  1. In the bottom-left Network Devices menu, click the Security icon (it looks like a brick wall).
  2. Find the model 5505 in the device list.
  3. Click and drag two 5505 ASAs into your workspace.

To place end devices:

  1. Go to the bottom-left main menu and click the End Devices icon (looks like a monitor/PC).
  2. Select the PC icon and drag two into the New York branch (left).
  3. Select the Server icon and drag two into the Tokyo branch (right).
  4. Select the Laptop icon and place it near the Internet router to act as the attacker.

By default, Packet Tracer numbers devices sequentially (e.g., Router0, Router1). To change the display name:

  1. Click directly on the text label below the device icon in the workspace.
  2. Type the new name (e.g., Switch 1, R1, Firewall 1, Attacker).
  3. Press Enter to save the name.

*Note: This changes the display name in the GUI, not the internal administrative hostname set via the CLI.

To cable the network without manually selecting Straight-Through or Crossover cables:

  1. Click the Connections icon (the lightning bolt) in the bottom-left menu.
  2. Click the gold lightning bolt icon ("Automatically Choose Connection Type").
  3. Click on the first device (e.g., PC1), then click on the second device (e.g., Switch 1). Packet Tracer will automatically select the correct cable type and available ports.
Speed Tip: If you hold down the CTRL key on your keyboard while clicking the gold lightning bolt, it will remain selected after you make a connection, allowing you to quickly wire the entire network without re-selecting the tool!
