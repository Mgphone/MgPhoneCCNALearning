Day 57: Wireless Security

WPA2, WPA3, 802.1X & EAP

🛡️

The Evolution of Encryption

Because wireless signals travel through the air, anyone with an antenna can capture your packets. Encryption is mandatory.

WEP (Wired Equivalent Privacy) - The Legacy Flaw

Used the RC4 cipher. Totally broken. The Initialization Vectors (IVs) were too short and reused often. Hackers can crack WEP in minutes by capturing enough packets. Never use it.

WPA (Wi-Fi Protected Access) - The Stopgap

A temporary fix for WEP that could run on older hardware. Introduced TKIP (Temporal Key Integrity Protocol) which dynamically changed keys. Still considered insecure today.

WPA2 - The Modern Standard

Required new hardware. Scrapped RC4/TKIP entirely and introduced AES (Advanced Encryption Standard) running in CCMP mode. It remains highly secure, provided you use a strong password.

WPA3 - The Future

The newest standard. Replaces CCMP with the even stronger GCMP mode. Makes Protected Management Frames (PMF) mandatory to stop deauthentication attacks. Dramatically improves public hotspot security (OWE).

🔑

Personal vs. Enterprise

Once you choose an encryption method (like WPA2), you must choose an Authentication method.

Personal (PSK)

Pre-Shared Key. Everyone connecting to the network types in the exact same password.

Problem: If an employee leaves, you must change the password on every single device in the company to keep them out.

Enterprise (802.1X)

Every user authenticates with their own unique username and password (or certificate), usually backed by Active Directory and a RADIUS server.

Benefit: You can instantly revoke one person's access without affecting anyone else.

🏛️ The 802.1X Framework

Enterprise Wi-Fi requires a robust authentication framework. 802.1X defines three specific roles.

💻

1. Supplicant

The end-user's device (laptop, phone) requesting access to the network.

📡

2. Authenticator

The Wireless LAN Controller (WLC) or the Access Point. It acts as the "bouncer," blocking traffic until identity is verified.

🗄️

3. Auth Server

The RADIUS server (e.g., Cisco ISE). Holds the actual database of valid credentials and tells the Authenticator "Yes" or "No".

📜 EAP Types

802.1X is just the framework. The actual conversation happens using the Extensible Authentication Protocol (EAP).

EAP-TLS (Most Secure)

Requires a digital certificate installed on the RADIUS server AND a certificate installed on every single client PC. Highly secure, but difficult to deploy.

PEAP (Most Common)

Protected EAP. Only requires a certificate on the RADIUS server. The server establishes an encrypted TLS tunnel, and the client securely passes their standard username/password inside the tunnel.

🚀 WPA3 Key Enhancements

SAE (Replaces PSK)

Simultaneous Authentication of Equals. In WPA2, a hacker could capture the 4-way handshake, take the packet home, and brute-force the PSK offline. WPA3's SAE uses a cryptographic "Dragonfly" handshake that completely prevents offline dictionary attacks.

PMF Mandatory

Protected Management Frames. In WPA2, an attacker can forge "deauthentication" frames to kick you off the network (causing a DoS) because management frames were unencrypted. WPA3 encrypts them, stopping this attack.

🪝 Brain Hooks

The 802.1X Nightclub

Supplicant: The guest trying to get in.
Authenticator: The Bouncer at the door (The AP).
Auth Server: The Manager inside with the VIP list (RADIUS). The Bouncer asks the Manager if the guest is allowed in.

PEAP = Passwords

If you are typing a Username and Password into a prompt to join Wi-Fi, you are almost certainly using PEAP.

⚡ Quick Quiz (Active Recall)

Click questions to reveal answers.

1. Which encryption standard and mode were introduced as the mandatory requirement for WPA2 certification? ▼
AES (Advanced Encryption Standard) running in CCMP mode. (WPA used TKIP, WEP used RC4).
2. If a WLAN is configured for "WPA2 Personal", what specific authentication method is being used? ▼
A Pre-Shared Key (PSK). All users share the exact same passphrase.
3. Which EAP type is considered the most secure but requires digital certificates to be installed on both the RADIUS server and every client machine? ▼
EAP-TLS (Transport Layer Security).
4. To prevent offline dictionary attacks, WPA3 replaces the Pre-Shared Key (PSK) exchange mechanism with what new protocol? ▼
SAE (Simultaneous Authentication of Equals).
