IPv4 Anatomy

An IPv4 address is a 32-bit logical address. It operates at Layer 3 (Network Layer) of the OSI model.

32 Bits Total 4 Octets 8 Bits per Octet

192.168.1.10

11000000.10101000.00000001.00001010

  • Computers read IP addresses in pure Binary (1s and 0s).
  • Humans read them in Dotted Decimal format.
  • The highest value a single octet can be is 255 (11111111).
🔢

The 8-Bit Binary Chart

You MUST memorize this chart to convert between Binary and Decimal quickly on the exam. Each bit represents a power of 2.

27 26 25 24 23 22 21 20
128 64 32 16 8 4 2 1
1 0 1 0 1 0 0 0

Example: 128 + 32 + 8 = 168

🗂️

IPv4 Classes

Historically, IP addresses were divided into classes based on the value of their first octet.

  • Class A 1 - 126
  • Class B 128 - 191
  • Class C 192 - 223
  • Class D (Multicast) 224 - 239
  • Class E (Experimental) 240 - 255

* Note: 127.x.x.x is reserved for loopback (testing).

🏠 Network vs. Host

Every IP address is split into two logical parts. This split is defined by the Subnet Mask.

1. Network Portion

Identifies the specific network segment (like a Street Name). All devices on the same LAN must have the identical Network portion.

2. Host Portion

Identifies the specific device on that network (like a House Number). Must be unique for every device on the LAN.

The Subnet Mask Rule:

Binary 1s = Network portion.
Binary 0s = Host portion.

🪝 Brain Hooks & Math Tricks

The Subtraction Method (Dec to Bin)

Start from left (128). Can you subtract 128 from your number? If YES, write a 1 and subtract it. If NO, write a 0 and move right. Repeat down to 1.

Class Ranges Mnemonic

A: 1-126 (Massive, half the internet)
B: 128-191
C: 192-223 (Smallest, most common for home)
Rule of thumb: Look at the first number before the first dot to know the class.

⌨️ CLI Basics

interface g0/0 Enter interface configuration mode.
ip address 192.168.1.1 255.255.255.0 Assigns the IP address AND Subnet Mask to the interface simultaneously.
no shutdown Powers up the interface.

1. How many bits make up an IPv4 address, and how many octets is it divided into? ▼
An IPv4 address is 32 bits long, divided into 4 octets (8 bits each).

170. (128 + 0 + 32 + 0 + 8 + 0 + 2 + 0)
3. What Class is the IP address 172.16.0.5? ▼
Class B. The first octet is 172, which falls within the Class B range of 128-191.
4. What configuration parameter dictates exactly where the Network portion ends and the Host portion begins? ▼
The Subnet Mask.
