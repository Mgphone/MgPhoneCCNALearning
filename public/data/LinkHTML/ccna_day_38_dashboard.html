Day 38: DNS

Domain Name System & Resolution

📖

What is DNS?

Humans memorize names (like google.com). Computers only understand IP addresses. DNS is the Application Layer (L7) protocol that translates between the two.

Fully Qualified Domain Name (FQDN)

An FQDN specifies the exact, complete name of a specific host on the internet (e.g., www.cisco.com.). Note the invisible dot at the very end representing the root!

TCP vs. UDP Port 53

  • UDP Port 53: Used for standard client queries. It's fast and lightweight.
  • TCP Port 53: Used for Zone Transfers (when DNS servers sync massive databases with each other) or if a query response is too large to fit in a single UDP packet (>512 bytes).
🌳

The DNS Hierarchy

DNS is a massive, distributed database spread across millions of servers globally. It is structured like an inverted tree.

1. Root Servers (.)

The absolute top of the tree. There are 13 logical root server clusters globally. They don't know where google.com is, but they know who manages the ".com" servers.

2. TLD Servers

Top-Level Domain servers manage extensions like .com, .org, .net, .edu. They point to the specific servers managing a requested domain name.

3. Authoritative Servers

These are the servers directly responsible for a specific domain (like cisco.com). They hold the actual IP address records for that domain.

🗂️ Common Record Types

DNS doesn't just store IP addresses. It stores different types of resource records.

  • A Record Maps a Name to an IPv4 address.
  • AAAA Record Maps a Name to an IPv6 address. (Quad-A).
  • CNAME Canonical Name. An alias pointing one name to another name (e.g., www to root).
  • MX Record Mail Exchange. Tells email servers where to deliver email for a domain.
  • PTR Record Pointer Record. Maps an IP back to a Name (Reverse DNS lookup).

🔄 The Resolution Process

When you type a URL, your PC doesn't do all the hard work itself. It delegates the job.

Recursive Query (The Client)

Your PC asks your local ISP's DNS Server (or 8.8.8.8): "I need the IP for cisco.com, and I demand you find the complete answer for me."

The Local DNS server does the heavy lifting.

Iterative Query (The Server)

The Local DNS server goes out to the internet and asks the Root Server. The Root replies with a referral: "I don't know, but ask the .com TLD server."

The Local DNS server iterates through the tree (Root -> TLD -> Authoritative) until it gets the exact IP.

🪝 Brain Hooks

The Global Phonebook

Nobody memorizes their friends' phone numbers anymore; they look up the name in their Contacts app. DNS is the Contacts app for the Internet.

AAAA = IPv6

An IPv4 address is 32 bits (A record).
An IPv6 address is 128 bits.
128 is 4x larger than 32, so IPv6 gets four A's (AAAA).

⌨️ Cisco DNS Configuration

Router(config)# ip name-server 8.8.8.8 1.1.1.1 Configures the router to use external DNS servers when you attempt to ping or resolve a hostname.
Router(config)# no ip domain-lookup Stops the router from treating CLI typos as hostnames. (Prevents the dreaded "Translating... domain server (255.255.255.255)" hang).
Router(config)# ip domain-name local.lab Sets the default domain name for the router. Required if you want to generate RSA crypto keys for SSH!
Router# show hosts Displays the default domain, the configured name servers, and the local DNS cache of the router.

🔧 Troubleshooting

Windows DNS Commands

nslookup: Opens a tool to manually query DNS servers and verify records.

ipconfig /displaydns: Views the PC's local DNS cache.

ipconfig /flushdns: Clears the cache. Essential if a website recently migrated to a new IP but your PC still tries the old one.

Can Ping IP, but not Name

Issue: `ping 8.8.8.8` works, but `ping google.com` fails.

Fix: Layer 3 routing is fine. The issue is purely DNS. Verify `ip name-server` is configured and not blocked by an ACL.

⚡ Quick Quiz (Active Recall)

Click questions to reveal answers.

1. A standard client DNS query uses UDP Port 53. What protocol and port number are used when two DNS servers need to perform a Zone Transfer? ▼
TCP Port 53. TCP provides the necessary reliability and sequencing for transferring massive amounts of database records.
2. Which type of DNS record maps a Fully Qualified Domain Name (FQDN) to an IPv6 address? ▼
An AAAA Record (Quad-A record).
3. You mistakenly type "shw ip route" in the Cisco CLI, and the router freezes, trying to contact a domain server. Which global configuration command prevents this annoying behavior? ▼
no ip domain-lookup
4. A web server's IP address changed last night, but your Windows PC is still trying to connect to the old IP address. Which Command Prompt command should you run to force the PC to retrieve the new IP? ▼
ipconfig /flushdns. This clears the local DNS cache, forcing the PC to send a fresh query to the DNS server.
