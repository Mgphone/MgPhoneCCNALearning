Day 52: LAN Architectures

Hierarchical Design, Spine-Leaf & SOHO

🏢

The 3-Tier Hierarchical Model

Cisco's classic blueprint for large enterprise networks. It divides the network into three modular layers to ensure scalability, high availability, and easy troubleshooting.

1. Core Layer (The Backbone)

The high-speed highway of the network. Its ONLY job is to switch traffic as fast as humanly possible.

  • Connects multiple Distribution layers together.
  • Must be highly available and redundant.
  • Rule: NO policies, NO ACLs, NO packet manipulation here. Anything that slows down switching is forbidden.

2. Distribution Layer (The Brains)

The aggregation point for Access switches and the boundary between Layer 2 and Layer 3.

  • Performs Inter-VLAN Routing.
  • Enforces security policies using Access Control Lists (ACLs).
  • Provides First Hop Redundancy (HSRP/VRRP).
  • Summarizes routes before sending them to the Core.

3. Access Layer (The Edge)

Where end devices (PCs, printers, IP phones, APs) actually connect to the network.

  • Provides Layer 2 switching.
  • Enforces edge security (Port Security, DAI, DHCP Snooping).
  • Provides power via PoE (Power over Ethernet).
  • Marks QoS values on traffic entering the network.

Collapsed Core (2-Tier)

A 3-tier architecture is overkill and too expensive for small-to-medium businesses. The solution is the Collapsed Core model.

How it Works

The Core and Distribution layers are merged into a single device (or a pair of redundant Multilayer switches).

  • Saves money on physical hardware.
  • Maintains the Access layer separately.
  • The Collapsed Core switch handles high-speed switching AND routing/ACLs simultaneously.

🍂 Spine-Leaf Architecture (Data Centers)

Modern data centers have massive amounts of "East-West" traffic (Server A talking to Server B), rather than "North-South" traffic (Server talking to the Internet). The 3-Tier model is too slow for East-West traffic. Spine-Leaf is the modern standard (used in Cisco ACI).

The Golden Rules of Spine-Leaf

  • Every Leaf connects to Every Spine.
  • Leaves NEVER connect to other Leaves.
  • Spines NEVER connect to other Spines.
  • Servers/Endpoints ONLY connect to Leaf switches.

The Benefit: Equidistance

Because of this full-mesh wiring, every Leaf is exactly ONE HOP away from every other Leaf.

This guarantees consistent, highly predictable latency for server-to-server communication, avoiding Spanning Tree blocked ports by using Layer 3 routing (like ECMP) on all links.

⚡ Power over Ethernet (PoE)

A key feature of the Access Layer. Allows a switch to send DC electrical power over the same UTP copper cable used for data.

PoE (802.3af)

Provides up to 15.4 Watts per port. Good for basic IP phones and standard APs.

PoE+ (802.3at)

Provides up to 30 Watts per port. Required for PTZ cameras and advanced Wi-Fi 6 access points.

UPoE / PoE++ (802.3bt)

Provides 60W to 90W. Enough to power compact switches, smart lighting, and thin-client PCs.

🪝 Brain Hooks

The Highway System

Access: Your driveway. Where you get onto the network.
Distribution: The highway On-Ramp. Enforces the rules (traffic lights/ACLs).
Core: The Interstate. No stoplights, no speed bumps, just pure speed.

Traffic Directions

North-South: Client to Server (or to Internet).
East-West: Server to Server (Back-end processing).

🏠 SOHO Architecture

In a Small Office/Home Office, you don't have racks of Core, Distribution, and Access switches. Instead, you use a collapsed "all-in-one" device (like a consumer Wi-Fi router or a small Meraki device).

Switch (LAN ports) Router (WAN port) Firewall (NAT) Access Point (WLAN) DHCP Server

SOHO networks prioritize simplicity, low cost, and ease of management (often cloud-managed) over extreme scalability and redundancy.

⚡ Quick Quiz (Active Recall)

Click questions to reveal answers.

1. In the traditional 3-Tier Hierarchical model, which specific layer is responsible for enforcing security policies via ACLs and performing Inter-VLAN routing? ▼
The Distribution Layer. It acts as the intelligent boundary between the Access layer and the Core layer.
2. What is the architecture model called when a network designer physically merges the Core layer and the Distribution layer into the same physical hardware? ▼
The Collapsed Core (or 2-Tier) model.
3. In a Spine-Leaf Data Center architecture, how many hops does it take for any Leaf switch to forward traffic to any other Leaf switch? ▼
Exactly ONE hop. Because every Leaf connects to every Spine, traffic goes Leaf -> Spine -> Destination Leaf, ensuring highly predictable latency.
4. Which Power over Ethernet (PoE) standard was introduced to provide up to 30 Watts of power per port, supporting heavier devices like PTZ cameras? ▼
PoE+ (IEEE 802.3at). (Standard PoE 802.3af only provides 15.4W).
