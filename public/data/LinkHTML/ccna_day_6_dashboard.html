Speed & Duplex

Dictates how data flows on the physical link.

  • Half-Duplex Devices can send OR receive, but not at the same time. Commonly found on legacy Hubs. Collisions are possible.
  • Full-Duplex Devices can send AND receive simultaneously. Standard for modern Switches. Collisions do not occur.
  • Auto-Negotiation Devices share their capabilities and automatically select the best mutual speed (10/100/1000) and duplex (Full over Half).
💥

CSMA/CD

Carrier Sense Multiple Access with Collision Detection. The legacy algorithm used to manage traffic in Half-Duplex networks.

1. Carrier Sense: Listen to the wire. Is anyone else talking?

2. Multiple Access: If the wire is quiet, send data.

3. Collision Detection: If two devices send at the exact same time, their signals collide and corrupt.

4. The Jam Signal: If a collision happens, devices send a "jam" signal so everyone knows.

5. Backoff Timer: Devices wait a random amount of time before trying again.

Note: CSMA/CD is NOT used on Full-Duplex links.

🌐

Network Domains

Collision Domain

An area where frames can collide if sent simultaneously.

  • Hub: 1 single collision domain for all ports.
  • Switch/Router: 1 collision domain per port (separates domains).

Broadcast Domain

An area where a broadcast frame (FF:FF:FF:FF:FF:FF) will be forwarded.

  • Hub/Switch: 1 single broadcast domain (by default).
  • Router: 1 broadcast domain per port (Routers stop broadcasts!).

🪝 Brain Hooks & Visualizing Domains

The Walkie-Talkie vs Phone

Half-Duplex: A walkie-talkie. If both press the button, you hear screeching (Collision). You must say "Over" to take turns.

Full-Duplex: A smartphone call. Both can talk and listen at the exact same time without interrupting the signal.

Switches Divide Collisions

A Switch is like a traffic cop at an intersection with dedicated lanes. Because every device gets its own dedicated "lane" (port), cars (frames) don't crash into each other.

Routers Break Broadcasts

When a PC screams "WHERE IS THE PRINTER?!" (Broadcast), the Switch echoes it everywhere. The Router hears it, acts like a soundproof wall, and says, "Keep that noise in your own room."

⌨️ CLI Commands: Interface Config

Switch(config-if)# speed {10 | 100 | 1000 | auto} Manually hardcodes the port speed or sets it to auto-negotiate (default).
Switch(config-if)# duplex {half | full | auto} Manually hardcodes the duplex or sets it to auto-negotiate (default).
Switch# show interfaces status Excellent summary command! Shows the port status, VLAN, Duplex, Speed, and Type. Prefixes `a-` (e.g., `a-full`) indicate the value was auto-negotiated.
Switch# show interfaces f0/1 Shows detailed interface stats. Look for "CRC errors", "collisions", or "late collisions".

🔧 Troubleshooting

Duplex Mismatch

Scenario: One side is set to Full, the other is set to Half.

Symptom: The link will stay up, but you will see massive packet loss, slowness, and late collisions / runts in show interfaces.

Auto-Negotiation Failure Rule

If a Cisco switch fails to auto-negotiate, it senses the speed but falls back to default duplexes:

  • 10 or 100 Mbps = Half-Duplex
  • 1000 Mbps (1 Gbps) = Full-Duplex

⚡ Quick Quiz (Active Recall)

Click questions to reveal answers.

1. A Router has 4 active interfaces connected to 4 different switches. How many Broadcast Domains exist? ▼
4 Broadcast Domains. Routers break up broadcast domains, so each active router interface represents a separate broadcast domain.
2. Why is the CSMA/CD algorithm disabled when a port operates in Full-Duplex? ▼
Because Full-Duplex uses dedicated wire pairs for transmitting and receiving. This makes simultaneous communication possible, meaning collisions simply cannot occur, rendering CSMA/CD unnecessary.
3. If you have a single 24-port Switch, how many Collision Domains are there? ▼
24 Collision Domains. A switch isolates collisions to each individual port (every port is its own collision domain).
4. Which concise CLI command allows you to view the speed, duplex, and auto-negotiation status of all ports on a switch? ▼
show interfaces status. (Remember: an "a-" prefix like "a-full" means it was auto-negotiated).
