The OSI Model

A conceptual framework used to understand how networks operate.

7. Application: Network services to apps (HTTP, FTP, DNS).
6. Presentation: Data formatting, encryption, compression (JPEG, ASCII, TLS).
5. Session: Establishes, maintains, and terminates connections.
4. Transport: End-to-end connections, reliability, ports (TCP, UDP).
3. Network: Logical addressing, routing (IPv4, IPv6, Routers).
2. Data Link: Physical addressing, error detection (MAC, Switches, Ethernet).
1. Physical: Media, signal, binary transmission (Cables, Hubs, Bits).
🌐

TCP/IP Suite

The actual protocol stack used on the Internet today. Modern networking uses the 5-Layer Updated TCP/IP Model.

  • L5 Application Maps to OSI Layers 5, 6, 7.
  • L4 Transport Maps directly to OSI Layer 4.
  • L3 Network Maps directly to OSI Layer 3.
  • L2 Data Link Maps directly to OSI Layer 2.
  • L1 Physical Maps directly to OSI Layer 1.
📦

Encapsulation & PDUs

As data moves down the stack, headers are added. The resulting block of data at each layer is a Protocol Data Unit (PDU).

  • L5-L7 PDU: Data
  • L4 PDU: Segment (Adds TCP/UDP Header)
  • L3 PDU: Packet (Adds IP Header)
  • L2 PDU: Frame (Adds MAC Header & Trailer)
  • L1 PDU: Bits (Sent over wire)

🔧 Troubleshooting

The OSI model is highly tested as a troubleshooting framework.

Bottom-Up

Start at Layer 1. Is the cable plugged in? Is there power? Move up to L2 (MAC/VLANs), then L3 (IP routing). Best for physical outages.

Top-Down

Start at Layer 7. Can you browse the web? If not, check L4 (firewall ports blocked?), then L3 (ping). Best for software issues.

Divide & Conquer

Start at Layer 3 (Ping). If Ping works, L1-L3 are fine—move up. If Ping fails, move down to L2/L1.

⌨️ CLI & The OSI Model

ping [ip-address] Tests Layer 3 (Network) connectivity end-to-end.
traceroute [ip-address] Traces the Layer 3 path packet takes via routers.
show mac address-table Verifies Layer 2 (Data Link) switch operations.

⚡ Quick Quiz (Active Recall)

Click questions to reveal answers.

1. What is the Protocol Data Unit (PDU) for Layer 4 (Transport layer)? ▼
Segment. (Remember: Some People Fear Birthdays).
2. During encapsulation, which layer is unique because it adds both a Header AND a Trailer to the data? ▼
Layer 2 (Data Link Layer). It adds the MAC Header and a Trailer containing the FCS (Frame Check Sequence) to detect errors.
3. At which OSI layer do Routers primarily operate to determine the best path for traffic? ▼
Layer 3 (Network Layer). Routers use logical IP addresses to make routing decisions.
4. Which layer of the OSI model is responsible for data encryption (e.g., TLS) and formatting (e.g., JPEG, ASCII)? ▼
The Presentation Layer (Layer 6). It ensures the data is in a readable format for the application layer and handles encryption/compression.
