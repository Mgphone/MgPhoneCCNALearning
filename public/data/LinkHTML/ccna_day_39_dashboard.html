Day 39: DHCP

Dynamic Host Configuration Protocol

🤖

What is DHCP?

Typing IP addresses manually on 500 computers is a nightmare. DHCP automates the assignment of IPv4 parameters to end devices.

What it Provides

A DHCP server typically leases four critical pieces of information to a client:

  • 1. IP Address
  • 2. Subnet Mask
  • 3. Default Gateway (Router)
  • 4. DNS Server(s)

UDP Ports 67 & 68

  • UDP Port 67: The Server listens on this port.
  • UDP Port 68: The Client listens on this port.
✉️

The DORA Process

When a PC boots up, it has no IP address. How does it get one? It initiates the 4-step D.O.R.A. process.

Discover

Client -> Server

The PC sends a Layer 2 Broadcast (FF:FF:FF:FF:FF:FF) shouting: "Are there any DHCP servers out there?"

Offer

Server -> Client

A server hears the broadcast and replies (usually Unicast) offering an available IP address from its pool.

Request

Client -> Server

The PC Broadcasts a message saying: "I accept the offer from Server X!" (Broadcasting this tells all other servers they can keep their offered IPs).

Ack

Server -> Client

The server sends an Acknowledgment finalizing the lease. The PC can now use the IP!

🌉 DHCP Relay (The Router Problem)

The Problem: The very first step of DHCP (Discover) is a broadcast. But routers block broadcasts! If the DHCP server is on a different VLAN/Subnet than the PC, the broadcast is dropped, and the PC never gets an IP.

The Solution: ip helper-address

You configure the router interface facing the PCs to act as a DHCP Relay Agent.

When the router hears the DHCP Discover broadcast, it intercepts it, converts it into a Unicast packet, and forwards it directly to the specific IP address of the remote DHCP server.

🪝 Brain Hooks

The Restaurant Analogy (DORA)

D: You walk in and shout, "Any waiters here?"
O: Waiter says, "I can offer you Table 5."
R: You shout, "I'll take Table 5 from Bob!" (So other waiters stop trying to seat you).
A: Bob says, "Great, here's your menu."

"Helpers Help Broadcasts"

Whenever you have a server (DHCP, TFTP) separated from clients by a router, remember that the ip helper-address is the magical bridge that turns local broadcasts into routable unicasts.

⌨️ Cisco DHCP Server Config

Step 1: Exclude IPs (Do this FIRST!) Router(config)# ip dhcp excluded-address 192.168.1.1 192.168.1.10 Prevents the server from handing out IPs already in use by printers, servers, or the router itself.
Step 2: Create the Pool Router(config)# ip dhcp pool LAN_USERS Router(dhcp-config)# network 192.168.1.0 255.255.255.0 Router(dhcp-config)# default-router 192.168.1.1 Router(dhcp-config)# dns-server 8.8.8.8 Defines the subnet block to lease from, the gateway, and DNS.
Verification Router# show ip dhcp binding Shows the MAC addresses of PCs and the IPs they currently hold.
The Helper Address (On the Interface facing the PCs) Router(config-if)# ip helper-address 10.0.0.50 Required only if the DHCP server is 10.0.0.50 and sits on a different network than this interface.

🔧 Troubleshooting APIPA

PC IP is 169.254.x.x

Issue: APIPA (Automatic Private IP Addressing). The PC yelled "Discover" but nobody answered.

Fixes:

  • Check if the DHCP Server service is running.
  • Check if the pool is completely out of IP addresses.
  • Most likely: You forgot the `ip helper-address` on the router interface, so the broadcasts are being dropped!

⚡ Quick Quiz (Active Recall)

Click questions to reveal answers.

1. What do the letters in the DHCP DORA process stand for? ▼
Discover, Offer, Request, Acknowledge.
2. If a DHCP server is placed in VLAN 100, but PCs in VLAN 10 need an IP address, what interface command must be applied to the router's VLAN 10 gateway? ▼
ip helper-address [Server-IP]. This acts as a DHCP Relay to convert the broadcast into a unicast packet.
3. You run `ipconfig` on a user's Windows machine and notice its IP address is 169.254.50.5. What does this instantly tell you? ▼
The PC failed to contact a DHCP server. It auto-assigned itself an APIPA (Link-Local) address.
4. DHCP relies on which Transport Layer protocol and which two specific port numbers? ▼
UDP Ports 67 (Server) and 68 (Client).
