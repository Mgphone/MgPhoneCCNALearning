NDP, SLAAC, EUI-64 & The IPv6 Header

📦

The Streamlined IPv6 Header

Despite IPv6 addresses being 4x larger than IPv4, the IPv6 header is vastly simplified for faster router processing. It is exactly 40 Bytes and fixed in size (no IHL field needed).

Version (4)
Traffic Class (8)
Flow Label (20)
Payload Length (16)
Next Header (8)
Hop Limit (8)
Source IPv6 Address (128 bits)
Destination IPv6 Address (128 bits)
  • Hop Limit: Replaces IPv4's TTL. Prevents loops.
  • Next Header: Replaces IPv4's "Protocol" field (e.g., TCP=6).
  • Traffic Class: Replaces IPv4's DSCP (QoS).
  • Removed: Header Checksum (Layer 2 & 4 already check for errors!).
  • Removed: Fragmentation fields (IPv6 routers do not fragment packets. Hosts must determine MTU size beforehand).
🧬

EUI-64 Format

Extended Unique Identifier (EUI-64) is a method for a device to automatically generate the host portion (last 64 bits) of its IPv6 address using its own MAC address.

The 3-Step Process:

  1. Take the 48-bit MAC address and split it directly in half.
    0011.22 | 33.4455
  2. Insert FFFE exactly in the middle to make it 64 bits.
    0011.22FF.FE33.4455
  3. Invert the 7th bit (usually turns a '00' into a '02').
    0211.22FF.FE33.4455

📡 NDP (ICMPv6)

IPv6 doesn't use ARP (because ARP uses broadcasts). Instead, it uses Neighbor Discovery Protocol (NDP), running on ICMPv6 and Multicast.

  • NS & NA (Replaces ARP)

    Neighbor Solicitation (NS): "Who has IPv6 address X? Send me your MAC." (Sent to a special Solicited-Node Multicast address).

    Neighbor Advertisement (NA): "That's me! Here is my MAC."

  • RS & RA (Router Discovery)

    Router Solicitation (RS): A PC boots up and asks: "Are there any routers on this link?"

    Router Advertisement (RA): The router replies: "I'm here! Here is your /64 subnet prefix and your Default Gateway IP."

🪄

SLAAC

Stateless Address Autoconfiguration allows a device to obtain an IPv6 address without needing a DHCP server.

1

PC boots up and sends an RS (Router Solicitation) message.

2

The Router replies with an RA containing the network's /64 Network Prefix.

3

The PC takes that Prefix and generates its own 64-bit Interface ID (either randomly or using EUI-64) to form a full 128-bit address.

DAD (Duplicate Address Detection)

Because the PC made up its own IP address, it must verify no one else is using it! Before using the IP, the PC sends an NDP Neighbor Solicitation (NS) for its own newly created IPv6 address. If nobody replies with an NA, the IP is safe to use.

🪝 Brain Hooks

The EUI-64 "FFFE" Sandwich

When you see FFFE smack in the middle of a massive IPv6 address's host portion, you immediately know it was auto-generated using EUI-64 from a MAC address.

SLAAC Constraint

SLAAC ONLY WORKS if the router advertises a /64 subnet. If the subnet is /60 or /80, SLAAC breaks, because the host always reserves exactly 64 bits to generate its Interface ID.

⌨️ IPv6 Configuration & Verification

Router(config-if)# ipv6 address 2001:db8:1::/64 eui-64 Assigns the network prefix, but tells the router to calculate its own host portion using its MAC address and the EUI-64 rules.
Router(config-if)# ipv6 address autoconfig Configures a router interface to act like a PC and use SLAAC to request an IPv6 address from another upstream router.
Router# show ipv6 neighbors The IPv6 equivalent of `show arp`. Lists known IPv6 addresses, their mapped MAC addresses, and their state (e.g., REACH, STALE).
Router# show ipv6 route Displays the IPv6 routing table. (Remember: `ipv6 unicast-routing` must be enabled globally for this to populate dynamically).

1. What is the name of the field in the IPv6 header that replaced the IPv4 TTL (Time To Live) field to prevent routing loops? ▼
Hop Limit.
2. Since IPv6 does not use ARP, which two specific NDP messages are used to resolve an IPv6 address to a MAC address? ▼
Neighbor Solicitation (NS) and Neighbor Advertisement (NA).
3. What mechanism does a host use during SLAAC to ensure the IPv6 address it just generated isn't already being used by someone else? ▼
DAD (Duplicate Address Detection). It sends an NS message for its own address.
4. When converting a 48-bit MAC address into a 64-bit Interface ID using EUI-64, what exact 4-character hex sequence is inserted into the middle? ▼
FFFE
