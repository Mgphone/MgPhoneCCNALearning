✈️ The Three Network Planes

Management Plane

Traffic used to access, configure, and manage the device.

  • SSH / Telnet
  • SNMP, Syslog
  • REST APIs

Control Plane

The "brains". Traffic used to build tables and learn where to send data.

  • Routing Protocols (OSPF, EIGRP, BGP)
  • STP (Spanning Tree)
  • ARP (Address Resolution Protocol)

Data (Forwarding) Plane

The "muscle". The actual process of moving packets from input to output.

  • Forwarding frames/packets
  • Encapsulation/Decapsulation
  • NAT, ACL Filtering

Traditional vs. SDN

Traditional Architecture

Distributed Control Plane: Every single router and switch runs its own routing protocols (OSPF, STP) to build its own routing/MAC tables. Device-by-device configuration.

SDN Architecture

Centralized Control Plane: The Control Plane is removed from individual devices and placed in a central SDN Controller. The controller computes routes and programs the Data Plane of all devices at once.

🧠 Memory Tricks & Analogies

The Taxi Dispatcher Analogy
Traditional: Every taxi driver uses their own map to decide routes (Distributed Control).
SDN: A central dispatcher (Controller) looks at the whole city's traffic and tells every taxi exactly where to turn (Centralized Control).

🎯 Active Recall: Quick Quiz

Test your knowledge. Click on the question to reveal the answer.

1. Which network plane is responsible for routing protocols like OSPF and EIGRP?
The Control Plane. It acts as the "brains" of the router, learning routes and populating the routing table, which is then used to program the Data Plane.
2. What is the primary difference between Northbound and Southbound APIs in an SDN architecture?
  • Northbound APIs: Connect the Controller "up" to Applications (using REST APIs).
  • Southbound APIs: Connect the Controller "down" to the Network Devices/Data Plane (using OpenFlow, NETCONF, etc.).
3. What is the name of the SDN Controller used in Cisco's Enterprise Campus solution (SDA)?
Cisco DNA Center. (Note: APIC is for Data Center/ACI, and vManage is for SD-WAN).
4. Which data plane function actually moves a packet from an ingress interface to an egress interface?
Forwarding. The data plane handles the actual moving of packets based on the rules and tables provided by the control plane.
5. True or False: In a purely traditional network architecture, a switch does not have a Control Plane, only a Data Plane.
False. In traditional networks, every router and switch contains its own distributed Control Plane (running STP, MAC learning, etc.). SDN is what abstracts the Control Plane away from the device.
